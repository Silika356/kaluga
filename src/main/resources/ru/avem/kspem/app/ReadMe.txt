Для изменения шаблона протокола редактируйте файл protocol.xlsx (Расширение должно быть .xlsx(Офис 2007 или старше)).
При возникновении ошибки удалите файл protocol.xlsx и сохраните любой протокол из базы данных в программе.

Справка:
"#PROTOCOL_NUMBER#" - Номер протокола
"#DATE#" - Дата начала испытания
"#TIME#" - Время начала испытания
"#OPERATOR#" - Оператор1 (Руководитель испытания)
"#OPERATOR2#" - Оператор2 (Испытатель)
"#SERIAL#" - Заводской номер объекта испытания
"#TESTITEM#" - Объект испытания(из базы данных)
"#TESTTYPE#" - Тип выбранного испытания
"#VALUES#" - С этой ячейки формируется таблица измеренных значений (Автоматически заполняется в 2 соседние ячейки, не объединять)
Ячейки со знаком '#' будут очищены
