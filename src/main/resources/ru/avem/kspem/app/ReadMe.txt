Для изменения шаблона протокола редактируйте файл protocol.xlsx (Расширение должно быть .xlsx(Офис 2007 или старше)).
При возникновении ошибки удалите файл protocol.xlsx и сохраните любой протокол из базы данных в программе.

Справка:
"#PROTOCOL_NUMBER#" - Номер протокола
"#DATE#" - Дата начала испытания
"#TIME#" - Время начала испытания
Ячейки со знаком '#' будут перезаписаны
